# Curso de Algoritmización
Curso elaborado en el año 2011 para impartirlo a mis contactos interesados en los temas de algoritmos y programación de concurso. Si bien el curso no pudo ser terminado, posee un grupo de conferencias, ejercicios y materiales adicionales que pueden ser de utilidad. Las conferencias abarcan temas básicos de la programación de concurso tales como: estructuras de datos básicas, técnicas de conteo, teoría de números y backtracking.

## Libros recomendados sobre el tema
- Art of Programming Contest
- Programming Challenges: The Programming Contest Training Manual
- Algorithm design practice for collegiate programming contests and education
- Algorithms For Competitive Programming
- Competitive Programming 3 The New Lower Bound of Programming Contests
- Guide to Competitive Programming Learning and Improving Algorithms Through Contests
